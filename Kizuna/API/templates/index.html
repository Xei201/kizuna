Общие данные:

Проект передаёт данные пользователей из сервиса для проведения вебинаров на платформу с CRM для обработки лидов. Реализованы опции автоматической передачи данных а также ручной загрузки. Автоматическая загрузка производится через API. Ручная загрузка возможна по нескольким сценариям: Через повторную акивацию уже проведённых сессий. Через загрузка списка сессий с внешнего сервиса и выбора сессии на перенос. Через загрузку CSV файла с данными. Также в качестве бонусных опций присутствует возможность загрузки любого списка пользователей на платформу CRM через CSV файл. А также микросервис по конвертированию данных тестирований в валидную форму.
